# Engineering

Debugging, pre-ship checklists, and codebase documentation.

- **[adoo](./adoo/SKILL.md)** — Structured problem-solving via the ADOO method (Assess, Deconstruct, Organize, Optimize) for bugs, refactors, and incidents where jumping straight to a fix is risky.
- **[pre-flight-check](./pre-flight-check/SKILL.md)** — Pre-commit / pre-deploy checklist for backend and full-stack changes: typecheck/lint gates, API response consistency, route ordering, schema checks, and result-checking. Ships as an adaptable template.
- **[codebase-artifact](./codebase-artifact/SKILL.md)** — Scan a codebase and produce a single self-contained HTML file documenting its architecture, user journey, component hierarchy, and key code patterns.
